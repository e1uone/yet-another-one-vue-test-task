## Установка и запуск

### Запуск в докере

```bash
yarn docker:build
yarn docker:run
```

### Установка зависимостей

```bash
yarn
```

### Запуск dev сервера

```bash
yarn dev
```

### Production сборка

```bash
yarn build
```

### Линтинг, форматирование и тайпчекинг

```bash
yarn lint
yarn format
yarn type:check
```
